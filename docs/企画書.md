# Markdown Quick Memo 企画書

## 目的

短いMarkdownメモを残すたびに高機能なエディタを起動する手間を減らし、Windowsショートカットから即座に入力を開始できる環境を提供する。

## 対象ユーザーと利用場面

対象は開発者本人とする。思いつき、後で確認する事項、作業中の一時メモを素早く記録する場面で使用する。

## 提供価値

- 小型画面をショートカットキーから起動できる。
- 入力欄と装飾表示を同一画面にまとめる。
- 一般的な `.md` ファイルとして保存し、他のエディタでも再利用できる。
- データベースや独自形式に依存しない。

## 対象機能

新規作成、読み込み、上書き保存、名前を付けて保存、検索、Undo/Redo、Markdown装飾、記号の省略表示、リンク操作、ローカル画像プレビュー、Windowsショートカット起動を対象とする。
